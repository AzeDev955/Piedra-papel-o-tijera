### ¿Cómo ejecutarlo?

1.  **Instalar todo por primera vez:**
2.  Tener creada en MYSQL la base de datos piedra-papel-tijeras
3. Nos ubicamos en la carpeta piedra-papel-tijeras
    
    Bash

    ```
    make init
    ```

    *(Esto instala los node_modules del back y del front, y rellena la base de datos)*.

4.  **Jugar (Arrancar Servidor + Cliente):**

    Bash

    ```
    make -j2 dev
    ```
